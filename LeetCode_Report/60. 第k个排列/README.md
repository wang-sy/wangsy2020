# LeetCode刷题记录——109. 有序链表转换二叉搜索树



## 题面

给出集合 [1,2,3,…,n]，其所有元素共有 n! 种排列。

按大小顺序列出所有排列情况，并一一标记，当 n = 3 时, 所有排列如下：

> "123"
> "132"
> "213"
> "231"
> "312"
> "321"


给定 n 和 k，返回第 k 个排列。

说明：

- 给定 n 的范围是 [1, 9]。
- 给定 k 的范围是[1,  n!]。


### 示例 1:

> 输入: n = 3, k = 3
> 输出: "213"

### 示例 2:

> 输入: n = 4, k = 9
> 输出: "2314"


## 分析

我们可以知道，当长度为1的时候有1种组合方案，当长度为2的时候有2种，当长度为n的时候有n!种。
那么我们可以参考二进制转换的原理，直接从高位进行检测，顺序检测即可。
